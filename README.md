# Lista 5 - Estrutura de Dados I (Java) 📑

Repositório destinado à resolução da **Atividade 5: TAD LISTAS** da disciplina de Estrutura de Dados I, do curso de **Tecnologia em Análise e Desenvolvimento de Sistemas** no **IFTM Campus Patrocínio**.

## 🚀 Sobre o Projeto

Esta lista foca na implementação de operações fundamentais de **Tipos Abstratos de Dados (TAD)** utilizando arrays, simulando o comportamento de listas dinâmicas e gerenciando o deslocamento manual de elementos na memória

### 🛠️ Tecnologias e Ferramentas
**Linguagem:** Java
**Instituição:** Instituto Federal do Triângulo Mineiro (IFTM)
**Docente:** Júnio Moreira

---

## 📋 Exercícios Desenvolvidos

| Questão | Descrição Técnica | Conceito Chave |
| :--- | :--- | :--- |
| **Questão 1** | Inserção com deslocamento para a direita[cite: 15, 17]. | Custo $O(n)$ em estruturas sequenciais. |
| **Questão 2** | Redimensionamento dinâmico de vetor (Dobro da capacidade). | Simulação de lista de tamanho variável. |
| **Questão 3** | Exclusão de elemento com *shift left* (deslocamento para esquerda). | Manutenção da continuidade dos dados. |
| **Questão 4** | Identificação e remoção de elementos duplicados. | Reorganização de coleção sem espaços vazios. |
| **Questão 5** | Fusão (*Merge*) de dois arrays já ordenados. | Comparação elemento a elemento via ponteiros. |

## 💻 Como Rodar o Projeto

O projeto foi estruturado para demonstrar a lógica de manipulação de vetores de forma didática.

1. Clone o repositório:
   ```bash
   git clone [https://github.com/moreiraptc/Lista5-Estrutura-de-Dados.git](https://github.com/moreiraptc/Lista5-Estrutura-de-Dados.git)
   ``
2. Abra o projeto no IntelliJ IDEA
3. Execute a classe Main.java
4. Escolha o número do exercício no console.

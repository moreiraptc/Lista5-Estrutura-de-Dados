# Lista 5 - Estrutura de Dados I (Java) 📑

[cite_start]Repositório destinado à resolução da **Atividade 5: TAD LISTAS** da disciplina de Estrutura de Dados I, do curso de **Tecnologia em Análise e Desenvolvimento de Sistemas** no **IFTM Campus Patrocínio**[cite: 7, 8, 14].

## 🚀 Sobre o Projeto

[cite_start]Esta lista foca na implementação de operações fundamentais de **Tipos Abstratos de Dados (TAD)** utilizando arrays, simulando o comportamento de listas dinâmicas e gerenciando o deslocamento manual de elementos na memória[cite: 14, 23].
